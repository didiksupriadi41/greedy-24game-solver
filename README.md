# greedy-24game-solver
Simple way to run the program:
1. Enter `python3 main.py` on Terminal
2. Enter 4 numbers.

##### Example 1
```
$ python3 main.py 
--> 1
--> 2
--> 3
--> 4
3 * 4 * 2 + 1
Score:  10
```
##### Example 2
```
$ python3 main.py 
--> 2 2 2 2
(2 + 2 + 2) * 2
Score:  0
```
